# Oregon Lottery Keno Scraper
---
A server to scrape the oregon lottery keno picks with a control api

## requirements
- node v12
- mongodb

## Setup
```bash
# Fill out env vars
cp .env.example .env

# Install deps
yarn

# Run dev server
yarn dev

# Run prod server
yarn start

```
